# frontend_cafeicultura
Frontend do projeto TECNOLOGIA E CAFEICULTURA - Solução Digital para a Gestão de Propriedades Rurais

Para o projeto funcionar:

### Pré-requisitos

* **Flutter SDK:** Certifique-se de ter o Flutter instalado e configurado corretamente no seu ambiente. Você pode verificar sua instalação com o comando:
    ```bash
    flutter doctor
    ```

* **Um Emulador/Dispositivo:** Você precisa ter um Emulador Android, Simulador iOS ou um dispositivo físico conectado e pronto para executar o app.



* **Instale as dependências:** 
```bash
flutter pub get.
```

* **Execute o aplicativo**: 
```bash
flutter run
```
